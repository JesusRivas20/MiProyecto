# Calculadora en Python

Este es un programa sencillo de calculadora desarrollado en Python. Permite realizar operaciones básicas.

## 🚀 Características
- Suma
- Resta
- Multiplicación
- División

## 🛠️ Tecnologías usadas
- Python

## ▶️ Cómo ejecutar el proyecto

1. Descargar el archivo `Calculadora.py`
2. Abrir en un editor como VS Code
3. Ejecutar el programa
